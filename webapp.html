About this project 

This project has benn developed as a console application in the first place, but when I finished it, I decided to rewrite the code to a windowed app with help of the PyQt5 package. It's a simple accounting app, with three features to household your money.

How does it work? 

Since accounting is a private thing where you don't want your informations to get spread, I had to write an encoding and a decoding mechanism for the JSON files where all the files are stored. I decided to use on OTP decoder (One time pad decoder) since it is a very ard to hack. The whole app is divided into three menues additional to the main menue. The development for this project is currently standing still but I plan to work on it in the future and maybe make it an Angular.js app
